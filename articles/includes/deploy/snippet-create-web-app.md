그런 다음, 봇을 게시할 봇 리소스를 만듭니다. 그러면 Azure에서 필요한 리소스가 프로비전되고 봇 웹앱이 만들어지며, 이 웹앱을 로컬 봇으로 덮어쓸 것입니다.

이 방법이 작업을 수행하는 유일한 방법은 아니지만, 정상적으로 작동하는 봇을 배포하는 데 필요한 모든 부분을 가장 쉽게 만들 수 있는 방법입니다.
